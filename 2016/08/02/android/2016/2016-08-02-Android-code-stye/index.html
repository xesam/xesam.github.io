用层次来构建人与机器的桥梁

通常一个 app 的数据流向是

Server <--> App <--> User

这个时候，程序员至少应该构建两层抽象，分别对应：

代码与接口之间的抽象：

Server <--> App

代码与需求之间的抽象：

App <--> User
