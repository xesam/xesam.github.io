一般来说上面的第二个参数都是传入一个函数句柄，但是大多数javascript程序员并不知道第二个参数可以传入一对象obj【DOM2接口 http://www.w3.org/TR/DOM-Level-2-Events/events.html#Events-EventListener 】，如此一来，当event执行的时候，会隐式的调用obj的handleEvent方法。
