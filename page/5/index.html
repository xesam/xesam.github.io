var & let & const

关于这个话题，我想要讨论的问题只有一个：什么时候应该用var，什么时候应该用 let，什么时候应该用const？

我的观点是，对于现代前端项目，有三个原则：

  1. 任何时候都不要用 var
  2. 能用 const 的地方都用 const
  3. 不能用 const 的地方，就用 let
