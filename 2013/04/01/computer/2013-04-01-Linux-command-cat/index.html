cat

来源单词：catenate（连接）

用途

主要包括下列用途：

  1. 将文本文件显示到屏幕
  2. 复制文本文件
  3. 连接文本文件
  4. 创建新的文本文件

命令参数

--help  显示帮助，几乎所有的GNU软件都有这么个同意的参数，另一个就是--version。

其他几个比较重要的是： 

-b, --number-nonblank    对非空输出行编号
-E, --show-ends          在每行结束处显示"$"
-n, --number             对输出的所有行编号
-s, --squeeze-blank      不输出多行空行
-T, --show-tabs          将跳格字符显示为^I
-v, --show-nonprinting   使用^ 和M- 引用，除了LFD和 TAB 之外

-A, --show-all           等于-vET
-e                       等于-vE
-t                       与-vT 等价
